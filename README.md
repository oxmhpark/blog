# 분류 방침

대분류를 제외한 나머지 범주는 복수설정이 가능하다.

| 범주 | 설정 | 템플릿 경로 |
|---|---|---|
| 대분류 | 행위의 종류<br>콜렉션(=폴더) 분리 | `대분류` 단락에서 기술 |
| 중분류 | 구체적인 게시 목적(=프로젝트)<br>`categories` 프론트매터 지정<br>개별 템플릿 작성 | [`/_archive_categories/`](https://github.com/oxmhpark/blog/tree/main/_archive_categories) |
| 소분류 | 구체적이고 반복인용 가능성이 있는 소재<br>`tags` 프론트매터 지정[^about-tags]<br>개별 템플릿 작성 | [`/_archive_tags/`](https://github.com/oxmhpark/blog/tree/main/_archive_tags) |
| 연도 | 게시물이 속하거나 취급하는 연도.<br>`years` 프론트매터 지정<br>개별 템플릿 작성 | [`/_archive_years/`](https://github.com/oxmhpark/blog/tree/main/_archive_years) |

[^about-tags]: 예를 들어, '인물'은 부적절하고 '시로 마사무네'는 적절하다.

# 운영 방침

* 첨부 파일, 사설, 토막글은 [메모장][memo]에 저장할 것.
* 문서 구조를 단순하게 유지하고 주요 갱신이력을 남길 것.
* 소통에 2차 비용을 치르지 말 것.
